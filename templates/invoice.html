Invoice Endpoint

In order to generate a invoice as pdf send a post request to the /invoice endpoint. Inside the body of the request you have to send the data for the invoice. The data has to be send as a json object of the following structure:

{
	title: "Invoice",
	senderName: "Codecool",
	senderAddress: ["Liechtensteinstra√üe 210", "1090 Wien"],
	recipientName: "Student No. 1",
	recipientAddress: ["Student Street 1", "1010 Wien"],
	items: [{name: "Fancy Coffee", amount: 10, price: "3.50"}, 
		{name: "Fancy Pants Coffee", amount: 5, price: "100000"}],
}

In order to use the API you need to set the X-API-TOKEN header with your api key. If you don't have a api key you can get one here
